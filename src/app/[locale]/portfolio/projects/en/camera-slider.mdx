# Summary

An automatic camera slider with a pan-tilt head for a parallax effect.

# Purpose

To provide more movement in a livestream presentation to increase viewer retention.

# Features

- Automated Control
- Pan-tilt mount
- Intuitive remote control
- Speed control
- Acceleration control
- Distance adjustment and angle calculations
- Automated software designed
- Custom PCB designs
- Custom modeled and CNCed
