# Summary

This is a tool that automates the process of finding and replacing text from a template to generate a set of printable documents.

# Purpose

To assist a school secretary in a specific task that was very time consuming and became quite rapid. Changing duration of task from around 2 hours to around 2 minutes.

# Features

- Bulk find and replace
- Specific date find and replace
- Easy student list manipulation
- Directly interfaces with Office API
- Outputs single PDF
- .NET application
