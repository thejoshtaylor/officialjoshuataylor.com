# Summary

Miniature scale earthquake table with a focus on accuracy and rigidity.

Watch a [test in action](https://youtu.be/DohGrMtXeWA).

# Purpose

For testing earthquake simulations on structures and soil at a 1:12 scale.

# Features

- Ball screw linear actuation
- Servo driver
- On board microcontroller sine wave generation
- Graphical PC interface for simple user control
- Acceleration test data acquisition and storage at high sample rates
- Safety mechanism for driver controlle
- Accelerations up to 0.7g
- Misumi extrusion table for ease of testing different objects
