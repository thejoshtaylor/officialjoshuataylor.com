# Summary

Does a bulk update on numerous AutoCAD files to bring drawing standards up to date.

# Purpose

To automate running scripts on many files that change standards and layouts.

# Features

- Rapid response
- Progress bar
- Formal report when finished
- Error report
- Special case scenario detection
- Dynamic command input
- Simple user interface
